# ⌨️ Guia de Atalhos: GNU Nano (v8.0+)

Este guia contém os comandos essenciais para produtividade no editor `nano`, configurado com o seu `.nanorc`.

---

## 🖱️ Navegação e Mouse
- **Scroll e Clique**: Habilitados via `.nanorc`. Você pode clicar para posicionar o cursor e usar a roda do mouse.
- **Início do Arquivo**: `Alt + \` (ou `Ctrl + Home`).
- **Fim do Arquivo**: `Alt + /` (ou `Ctrl + End`).
- **Ir para Linha/Coluna**: `Ctrl + _` (underline).

## ✂️ Seleção e Edição (Copiar/Colar)
| Ação | Atalho | Descrição |
| :--- | :--- | :--- |
| **Marcar/Selecionar** | `Alt + A` | Inicia a seleção (mova as setas para destacar o texto). |
| **Copiar** | `Alt + 6` | Copia a área marcada para o buffer. |
| **Recortar (Cortar)** | `Ctrl + K` | Remove a linha atual ou a área selecionada. |
| **Colar** | `Ctrl + U` | Cola o conteúdo que foi copiado ou recortado. |
| **Selecionar Tudo** | `Alt + \` depois `Alt + A` depois `Alt + /` | Vai ao topo, marca, e vai ao fim. |
| **Apagar Tudo** | (Após Selecionar Tudo) + `Ctrl + K` | Limpa o arquivo inteiro instantaneamente. |

## 🔍 Busca e Substituição
- **Buscar Texto**: `Ctrl + W` (Where is).
- **Próxima Ocorrência**: `Alt + W`.
- **Substituir**: `Ctrl + \` (ou `Alt + R`).

## 💾 Gestão de Arquivos
- **Salvar (Gravar)**: `Ctrl + O` (Confirme com `Enter`).
- **Sair**: `Ctrl + X` (Se houver mudanças, ele perguntará se deseja salvar).
- **Inserir outro arquivo**: `Ctrl + R`.
